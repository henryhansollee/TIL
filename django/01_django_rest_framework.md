# 01\_Django\_REST\_framework

### 1. 프로젝트 시작

#### 1-1. 가상환경

* 가상환경 생성

```text
$ python -m venv env
```

* 가상환경 시작

```text
$ source env/Scripts/activate
```

* 가상환경 종료

```text
$ deactivate
```

#### 1-2. Django & Django REST framework 설치

```text
$ pip install django
$ pip install djangorestframework
```

#### 1-3 Django 프로젝트 생성

```text
$ django-admin startproject backend .
```
