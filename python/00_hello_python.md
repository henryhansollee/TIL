# 00\_HELLO\_PYTHON

### 1. 파이썬 문법

#### 1-1. 인덴트\(Indent\)

* 공백 4칸 들여쓰기가 원칙이다.

#### 1-2. 네이밍 컨벤션\(Naming Convention\)

* 변수명, 함수명: 스네이크 케이스

#### 1-3. 타입 힌트

```text
def fn(a: int) => bool:
    ...
```

#### 1-4. 리스트 컴프리헨션\(List Comprehension\)

* 기존 리스트를 기반으로 새로운 리스트를 만들어내는 구문으로, 파이썬 2.0부터 지원되었으며, 하스켈 같은 함수형 언어에서 기능을 차용해온 파이썬의 대표적인 기능이다.

```text
[n * 2 for n in range(1, 10 + 1) if n % 2 == 1]
```
