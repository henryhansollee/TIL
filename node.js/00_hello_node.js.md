# 00\_HELLO\_NODE.JS

### 1. Node.js

#### 1-1. 서버

* 네트워크를 통해 클라이언트에 정보나 서비스를 제공하는 컴퓨터 또는 프로그램이다.
* 클라이언트란 요청을 보내는 주체로 브라우저일 수도 있고, 데스크톱 프로그램일 수도 있고, 모바일 앱일 수도 있고, 다른 서버에 요청을 보내는 서버일 수도 있다.

#### 1-2. 자바스크립트 런타임

* 런타임은 특정 언어로 만든 프로그램들을 실행할 수 있는 환경이다.

#### 1-3. 이벤트 기반

* 이벤트가 발생할 때 미리 지정해둔 작업을 수행하는 방식을 의미한다.
