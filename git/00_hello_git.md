# 00\_HELLO\_GIT

### Git config 설

```text
$ git config --list

# Global
$ git config --global user.name <유저이름>
$ git config --global user.email <유저이메일>

# Local
$ git config --local user.name <유저이름>
$ git config --local user.email <유저이메일>
```

### 새로운 저장소 생성

```text
$ git init
```

### 저장소 복제

```text
$ git clone
```

### 추가

```text
$ git add <파일명>
$ git add .
$ git add *
```
